# 弹弹play for iOS

[弹弹play](http://www.dandanplay.com/)的iOS版 

## 功能介绍
1. 视频智能识别
2. 支持从其它APP或者通过HTTP的方式导入视频

## 系统要求
最低支持 iOS 8.0
